3. Create `README.md` at repo root with the following sections IN THIS ORDER (per D-05, D-06, REL-01):

**Section 1: Product overview**
- H1 header: `# Toggl Import`
- Tagline: "Import Toggl Track time entries into your Obsidian daily notes with a single command."
- Core value paragraph: explain that the active note's filename (yyyy-mm-dd) determines the date, entries are fetched from Toggl's API, and formatted output is inserted at the cursor.
- Screenshot: `![Toggl Import demo](assets/demo.png)` (per D-07, D-08)
- Brief feature bullets:
  - Three output formats: Markdown table, plain text (configurable delimiter), or custom template (`$variable` placeholders)
  - Configurable columns: Description, Start time, Duration, Tags, Project
  - Sort entries by start time (ascending or descending)
  - Secure API token storage (device-local, not synced)
  - Test connection button to verify API token

**Section 2: Manual installation** (per D-05)
- H2 header: `## Manual Installation`
- Numbered steps:
  1. Download `main.js`, `manifest.json` from the latest release on GitHub
  2. Create folder `<vault>/.obsidian/plugins/obsidian-toggl-import/`
  3. Copy `main.js` and `manifest.json` into that folder
  4. Open Obsidian Settings > Community Plugins > enable "Toggl Import"

**Section 3: BRAT installation** (per D-06)
- H2 header: `## Install via BRAT`
- Brief steps:
  1. Install the [BRAT](https://github.com/TfTHacker/obsidian42-brat) plugin
  2. In BRAT settings, click "Add Beta Plugin"
  3. Paste `theaspect/obsidian-toggl-import`
  4. Enable "Toggl Import" in Community Plugins

**Section 4: Usage walkthrough**
- H2 header: `## Usage`
- Steps:
  1. Open Settings > Toggl Import, paste your Toggl API token (found at https://track.toggl.com/profile under "API Token"), click Test to verify
  2. Open (or create) a daily note whose filename starts with a date in `yyyy-mm-dd` format (e.g. `2026-04-16.md` or `2026-04-16 Wednesday.md`)
  3. Place cursor where you want entries inserted
  4. Open command palette (Ctrl/Cmd+P), run "Toggl Import: Import Toggl Entries"
  5. Time entries for that date appear at the cursor

**Section 5: Settings reference**
- H2 header: `## Settings`
- Table or definition list of all settings:
  | Setting | Description | Default |
  |---------|-------------|---------|
  | Toggl API token | Your Toggl Track API token | (empty) |
  | Test connection | Verify token against Toggl API | - |
  | Output format | `Markdown table`, `Plain text`, or `Custom template` | Markdown table |
  | Delimiter | Separator for plain text mode | `\|` |
  | Template | Template string with `$description`, `$start`, `$duration`, `$tags`, `$project` placeholders | `$description ($duration)` |
  | Sort order | Ascending (oldest first) or Descending (newest first) | Ascending |
  | Columns | Toggle Description, Start time, Duration, Tags, Project (disabled in template mode) | Description, Start time, Duration enabled |

**Section 6: Development setup**
- H2 header: `## Development`
- Commands:
  ```
  git clone https://github.com/theaspect/obsidian-toggl-import.git
  cd obsidian-toggl-import
  npm install
  npm run dev      # watch mode
  npm run build    # type-check + production bundle
  npm test         # run tests
  ```
- Note: copy `main.js` and `manifest.json` to your vault's plugin folder for testing. The [Hot-Reload](https://github.com/pjeby/hot-reload) community plugin auto-reloads on change.

**Section 7: License** (brief, at bottom)

- One line: `[MIT](LICENSE)`
